ℹ️ About Astravon Live Arena

AI-powered crowd intelligence, event safety and emergency response platform.

Project Overview

Astravon Live Arena is an intelligent event monitoring platform that combines Computer Vision, Artificial Intelligence, environmental monitoring, crowd analytics and emergency response into a unified command dashboard.

Core Features

  • 🎥 Real-time object detection
  • 👥 Crowd counting
  • 📈 Occupancy monitoring
  • 🌡️ Heat index prediction
  • 🚨 AI risk assessment
  • 🗺️ Emergency response visualization
  • 📊 Live dashboard analytics

Technology Stack

Frontend HTML5 • CSS3 • JavaScript
Backend FastAPI
AI Engine Python • YOLO • OpenCV
Communication REST API • WebSockets
Charts Chart.js
