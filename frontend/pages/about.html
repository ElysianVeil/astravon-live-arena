ℹ️ About Astravon Live Arena

AI-powered crowd intelligence, event safety and emergency response platform.

Project Overview

Astravon Live Arena is an intelligent event monitoring platform combining Computer Vision, Artificial Intelligence, environmental monitoring, crowd analytics and emergency response into one unified command dashboard.

Core Features

  • 🎥 Real-time Object Detection
  • 👥 Crowd Counting
  • 📈 Occupancy Monitoring
  • 🌡 Heat Index Prediction
  • 🚨 AI Risk Assessment
  • 🗺 Emergency Response Visualization
  • 📊 Live Dashboard Analytics
